# 飞书路由示例

## 示例 1：项目想法

输入：想做一个新的团队 onboarding 优化，下周先讨论方向。

输出：

- 当前请求类型：项目想法
- 当前主要目标：收敛新项目方向
- 推荐去向：GitHub 项目级 issue

## 示例 2：实现请求但缺工件

输入：把上次说的 onboarding 再优化一下，顺便看看 loading 状态。

输出：

- 当前请求类型：切片请求
- 当前主要目标：不明确
- 是否多个目标：是
- 推荐去向：继续澄清

## 示例 3：验收结论

输入：这版切片结果发我看下，确认后准备继续下一个。

输出：

- 当前请求类型：验收
- 当前主要目标：判断当前切片是否通过验收
- 推荐去向：人类审批

# 综合待办模板示例：家庭周末出游

这份文档只用于演示怎样使用 `docs/todo-template.md`。

## 主题

安排一次家庭周末出游

## 1. 协作方式规划

- [ ] 明确这次出游的目标：放松、陪伴孩子、还是顺便打卡
- [ ] 明确这次只安排一个周末，不顺便规划长假旅行
- [ ] 明确参与人、出发地和大致时间窗口
- [ ] 人类确认：这次周末出游是否值得进入筹备

## 2. 能力规划与验收规划

- [ ] 拆出关键事项：目的地、交通、住宿、预算、行程安排
- [ ] 明确每项的边界：预算上限、单日车程上限、孩子是否适合
- [ ] 明确验收标准：行程不赶、预算可接受、家人都能接受
- [ ] 明确哪些事可以先查资料，哪些事必须由家人拍板
- [ ] 人类确认：预算、目的地倾向和是否接受备选方案

## 3. 能力建设与术语补齐

- [ ] 查询并整理候选地点、交通方式和住宿选项
- [ ] 形成一版简单行程表和预算表
- [ ] 补齐家里需要准备的物品清单：衣物、零食、药品、充电器
- [ ] 若出现新约束，及时补进清单，例如“老人不爬山”“孩子午睡要保留”
- [ ] 人类 review：确认这版安排是否真的适合全家

## 4. 验收效果

- [ ] 检查这版安排是否真的能让周末出行更轻松
- [ ] 检查是否还有明显风险：太赶、太贵、太累、天气不稳
- [ ] 记录已知限制，例如“热门酒店订不到”“返程可能堵车”
- [ ] 记录下一步是直接预订，还是先换一个目的地
- [ ] 人类验收：决定通过、调整后重试或暂缓

## 5. 使用说明更新

- [ ] 把最终行程发给家人，确保所有人知道时间和安排
- [ ] 把需要携带和出发前要做的事整理成一页清单
- [ ] 删除已经作废的备选方案，避免临出发时混乱
- [ ] 确保家人只看到最终有效版本，而不是一堆草稿
- [ ] 人类确认：这份出游说明是否已经足够清楚

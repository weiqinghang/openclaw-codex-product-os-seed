# 验证记录台账

## 使用说明

用于记录 `龟丞相` 在真实飞书请求中的稳定性验证证据。

每条记录建议覆盖：

- 场景与输入
- 主请求分类与次级标签
- 读取动作
- 输出质量（四段式）
- 边界合规（是否越权）
- 升级路径（是否正确升级到人类 / Codex）
- 人工纠偏次数
- 工件回写完整率
- 结论与改进项
